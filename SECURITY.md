# Security Policy

Please report potential security issues privately.

## How to report

- Do **not** post vulnerability details in public Issues or Pull Requests.
- If private vulnerability reporting is available on GitHub for this repository, please use that channel.
- Otherwise, contact the maintainer through a private support or contact channel associated with the RunProof service.

## Notes

- This repository is a **public surface** only.
- The private source-of-truth repository is not publicly accessible.
- Do not include secrets, API keys, payment data, or personal data in reports.
- Response is provided on a best-effort basis.
